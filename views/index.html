Making it work

  1. npm install angular-translate --save
  2. In your Gruntfile.js:
    link to angular-translate.min.js and angular-translate-loader-url.min.js
  3. In your html file add the things to translate:
    <h1>{{'THING1' | translate}}</h1>
    <p>{{'THING2' | translate}}</p>
  4. In your html link to angular and angular-translate with angular first:
    <script type="text/javascript" src="/vendor/angular/angular.min.js"></script>
    <script src="/vendor/angular-translate/dist/angular-translate.min.js"></script>
    <script type="/vendor/angular-translate-loader-url/angular-translate-loader-url.min.js"></script>
    Also link to your own app.js file
  5. Actually set up your app in your app.js file:
    var myApp = angular.module('myApp', ['pascalprecht.translate']);
  6. Define your translation variables (i.e. what to translate):
    var transEN = {
    'THING1': "stuff",
    'THING2': "more stuff"
    };
    var transDE = {
    'THING1': "zeug",
    'THING2': "mehr"
    }
